# CarBasix Landing (Astro + Tailwind)

## Setup

1. Install dependencies:
   `npm install`
2. Configure analytics:
   copy `.env.example` to `.env` and set `PUBLIC_YANDEX_METRICA_ID`
3. Start dev server:
   `npm run dev`

## Build

`npm run build`

## Stack

- Astro
- Tailwind CSS
- Yandex Metrica (optional via env var)
