## AI Context: CarBasix Landing Page

**Project:** CarBasix
**Type:** B2B SaaS / vehicle import intelligence platform
**Audience:** vehicle importers, resellers, car traders, brokers, sourcing teams, especially in CIS-related import workflows
**Goal of the landing page:** convert visitors into free-trial users and demo leads
**Primary CTA:** “Try it free”
**Secondary CTA:** “Book a demo” or “See how it works”

**Positioning:**
CarBasix is a working platform, not a concept. It helps professionals search vehicles across multiple international marketplaces and auctions in one interface, filter by key parameters, compare options quickly, and estimate the full landed/import cost.

**Core value proposition:**
Instead of manually checking many marketplaces, comparing tabs, and calculating costs in spreadsheets, users can do this in one product workflow.

**Main product capabilities to present on landing page:**

* multi-marketplace vehicle search
* filters by marketplace, make, model, price, year, mileage, etc.
* listing comparison in one interface
* automatic landed/import cost estimation
* ability to mark/save/review listings
* links to original listing sources

**Design direction:**
premium SaaS, product-first, modern dashboard aesthetic, clean and trustworthy, not a car dealer website, not flashy automotive styling, more analytics/workflow oriented.

**Framework:** Astro
**Styling:** Tailwind CSS
**Analytics:** Yandex Metrica, optionally with a second analytics layer
**Hosting:** start with free static hosting (Cloudflare Pages or GitHub Pages), optional self-hosted VPS later
**Visual priority:** real product UI screenshots and product workflow visuals, not stock car photography
**Page objective:** explain value fast, show workflow, create trust, push free-trial signup.
